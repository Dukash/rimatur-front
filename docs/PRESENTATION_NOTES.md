# Notas para apresentação

Objetivo: preparar documentação clara para apresentar o projeto (pitch técnico).

Estrutura sugerida para apresentação (5-8 slides)

1. Título e contexto — objetivo do projeto e problema que resolve.
2. Arquitetura e stack — Next.js, TypeScript, CSS (Tailwind/Bootstrap), estrutura `app/`.
3. Principais rotas e fluxos — login, registro, mensagens (fluxos de usuário).
4. Componentes reutilizáveis — `Navbar`, formulários e padrões de UI.
5. Como rodar localmente — comandos principais.
6. Próximos passos e melhorias.
7. Perguntas.

Dicas

- Tenha links diretos para as páginas em `docs/` durante a apresentação.
- Prepare capturas de tela das telas principais.
- Se possível, um pequeno demo ao vivo com `npm run dev`.
