# Setup e Execução

Este documento descreve como configurar e executar o frontend localmente.

Pré-requisitos

- Node.js (>= 18 recomendado)
- npm

Instalação

```bash
npm install
```

Modo Desenvolvimento

```bash
npm run dev
```

Build para produção

```bash
npm run build
npm run start
```

Lint e formatação

```bash
npm run lint
npm run format
```

Notas

- O projeto usa Next.js (App Router). As rotas estão dentro de `app/`.
